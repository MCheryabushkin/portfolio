Радиопоглощающие материалы

Описание товара

Радиопоглощающий материал (РПМ) серии PF - ультра-широкополосные поглотители, специально предназначенные для применений в микроволновом диапазоне. PF поглотители имеют традиционную остроконечную пирамидальную форму. Оптимизация производственного процесса позволяет достигнуть полной пропитки диэлектрического резистивного материала и огнезащитных веществ. PF поглотители рекомендованы для использования с антеннами и для измерения ЭПР в безэховых камерах, обеспечивая очень высокую отражательную способность в микроволновом диапазоне частот. Данные поглотители из углеродного пенопласта также имеют превосходные характеристики при наклонном падении с низким уровнем обратного рассеивания.

Области применения

  • Измерение ЭПР
  • Измерение направленности антенны
  • Телекоммуникация/беспроводная связь
  • Спутниковая связь
  • Автомобильная промышленность
  • Военная промышленность
  • Авиационная промышленность
